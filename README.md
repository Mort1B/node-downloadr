node-downloadr

Downloads a whole transaction from arweave.

To run:

***npm install***

***node downloader.js --transaction <tx_id> --output <file_name>***

Example:

***npm install***

***node downloader.js --transaction "K9u_6E9tO8yr6Jx1D_lHz2tRhSIThPrXpmgazFw8BWI" --output "./bas.txt"***
